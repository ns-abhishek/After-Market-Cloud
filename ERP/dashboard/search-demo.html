How to Use the Enhanced Search

  • Type in the search box above or press Ctrl+K to focus
  • Search across all menu items, dropdown options, and bookmarks
  • Use the filter buttons to narrow down results by category
  • Navigate results with ↑ ↓ arrow keys
  • Press Enter to select or Esc to close
  • Recent searches are automatically saved and suggested
